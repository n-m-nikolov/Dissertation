What is Dependency Parsing?

Dependency Parsing is basically the process of parsing natural language to a Dependency Tree. There are two ways of achieving this goal â€“ by using constituency grammar or dependency grammar. The main difference between the two is that in dependency grammar for every word or morph in the sentence there is exactly one single node corresponding to it, where as in constituency grammar we can have one or more nodes corresponding to the same word or morph. Here we will be using only Dependency Grammar.
